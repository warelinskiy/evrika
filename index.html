Понимание ключевого слова `final`

В Java ключевое слово final является модификатором без доступа для классов, атрибутов и методов, что делает их неизменяемыми.

Main.java 
public class Main {
  final int x = 10;

  public static void main(String[] args) {
    Main myObj = new Main();
    // myObj.x = 25; will generate an error
    System.out.println(myObj.x);
  }
}

Ключевые характеристики:

  • check_circle Переменные: становится константой. Вы не можете переназначить значение.
  • check_circle Методы: запрещает переопределение метода в подклассах.
  • check_circle Классы: запрещает наследование класса.
